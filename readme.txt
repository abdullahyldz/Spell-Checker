How to Run 
1.	This project is written on Jupyter Notebook. Open a Jupyter Environment

2.	At the head of the notebook there is a section with file names. 
Enter which file you want to read.
Make sure files are at the same workspace with .ipynb file.

corpusFile : used to create the corpus
spellErrorsFile = used to create confusion matrices.
testCorrectFile = used to check correctness of the predictions
testMisspelledFile = used to test the program

3.	Go to Kernel > Restart & Run All
